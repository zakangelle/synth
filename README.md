## Develop

Install library dependencies:

```
$ npm install
```

Boot up the example app at [http://localhost:3000](http://localhost:3000) (implicitly rebuilds on code changes):

```
$ npm start
```

## Test

Run tests (implicitly re-runs on code changes):

```
$ npm test
```

Generate test coverage:

```sh
$ npm run coverage
```
